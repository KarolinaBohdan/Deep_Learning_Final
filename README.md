# Green Patent Detection Project

## Overview
The Green Patent Detection project aims to identify and analyze patents related to environmentally sustainable technologies. The project includes various assignments that explore different aspects of green patents.

## Table of Contents
1. [Assignments](#assignments)
2. [Installation](#installation)
3. [Usage](#usage)
4. [Technical Details](#technical-details)

## Assignments
1. **Assignment 1:** Define green patents and identify a set of relevant patents.
2. **Assignment 2:** Analyze the trends in green patents over the last decade.
3. **Assignment 3:** Develop a classification model to categorize green patents based on their technology area.
4. **Assignment 4:** Evaluate the impact of green patents on innovation and sustainability.

## Installation
To install the necessary dependencies, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/KarolinaBohdan/Deep_Learning_Final.git
   cd Deep_Learning_Final
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the project, use the following command:
```bash
python main.py
```

Make sure to adjust the configuration files according to your needs before running the script.

## Technical Details
- **Programming Languages:** Python
- **Libraries Used:** TensorFlow, Keras, Pandas, NumPy, Matplotlib
- **Data Sources:** U.S. Patent and Trademark Office database, academic journals, and other related databases for environmental technologies.
